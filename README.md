# 컴홈페이 (샘플) — Glassmorphism 스타일 지급결제 회사 홈페이지

로컬에서 간단히 미리보기하려면 저장된 `index.html` 파일을 브라우저로 열면 됩니다.

빠른 실행 (터미널에서):

```bash
# 루트에서 (이 리포지토리)
xdg-open index.html  # Linux
# 또는
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 열기
```

파일:

- [index.html](index.html) — 메인 페이지
- [styles.css](styles.css) — 스타일

원하시면 디자인(색상, 문구), 회사 로고, 실제 연락처를 반영해 수정해 드리겠습니다.
